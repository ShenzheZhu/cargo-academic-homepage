Place your images in this directory.

Suggested files:
- favicon.png       — Browser tab icon (32x32 or 64x64)
- placeholder.png   — Default image for publications and home page
- Any paper figures  — Place in asset/paper_fig/ subdirectory

Image tips:
- Publication figures display at max-width 400px, shown in grayscale by default
- Home page figure displays at max 50% width (380px max)
- All images are responsive on mobile
